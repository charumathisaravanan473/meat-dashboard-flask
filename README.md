 Meat Quality Detection Dashboard

A sensor-based IoT and Machine Learning project that detects whether meat is Fresh or Warning or Spoiled using real-time sensor data.

 Features

- Real-time sensor monitoring
- Meat quality prediction
- Live dashboard
- Machine Learning classification using Random Forest

 Technologies Used

- ESP32
- MQ135 Gas Sensor
- pH Sensor
- DHT22 Temperature & Humidity Sensor
- Python
- Flask
- Streamlit
- Random Forest
- Pandas
- Scikit-learn

 Sensor Parameters

- pH
- Gas (MQ135)
- Temperature
- Humidity

 Machine Learning

- Algorithm: Random Forest
- Input: Sensor readings
- Output: Fresh / Spoiled/Warning

Dataset

- 5000 sensor data records
- CSV format
- Binary classification (Fresh / Spoiled/ Warning)

 How to Run

1. Clone this repository.
2. Install the required libraries.
3. Run the dashboard.
4. View the live sensor readings and prediction.
